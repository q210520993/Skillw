Package org.bukkit.block.data

各种方块的 BlockData 类.

See: Description

  • Interface Summary 
    Interface Description
    Ageable'age' 表示农作物的不同生长阶段.
    AnaloguePowerable'power' 值代表当前通过此方块发出/传输红石信号的强度.
    Attachable'attached' 值表示绊线钩或绊线是否构成了一个完整的绊线回路并准备好触发.
    Bisected'half' 值表示该方块是一个两格高方块的哪一部分.
    BlockData
    Brushable'dusted' 表示方块被刷子清除的程度，即露出的程度。
    Directional'facing' 代表方块的朝向.
    FaceAttachable'face' 值代表一个拉杆或按钮固定在哪一个面上.
    Hangable'hanging' denotes whether the lantern is hanging from a block.
    Hatchable'hatch' 值为这些蛋中孵化出实体的数量.
    Levelled'level' 值代表该方块中液体的高度, 其液体要么单独存在要么在炼药锅内.
    Lightable'lit' 表示此方块 (红石或熔炉) 是否在发光 ———— 不是烧毁他们的意思.
    MultipleFacingThis class encompasses the 'north', 'east', 'south', 'west', 'up', 'down' boolean flags which are used to set which faces of the block textures are displayed on.
    Openable'open' 表示此方块是否开着.
    Orientable'axis' represents the axis along whilst this block is oriented.
    Powerable'powered' 数据值表示该方块是否被充能.
    Rail'shape' 数据值表示此铁轨的形态 (铁轨连接方向).
    Rotatable'rotation' 表示方块的旋转方向.
    Snowable'snowy' 属性表示此方块是否被雪覆盖 (通常情况下时因为 "它" 的头顶下着雪).
    Waterlogged'waterlogged' 表示方块是否处于流体之中.
  • Enum Summary 
    Enum Description
    Half垂直方向上一个方块的二分部分.
    AttachedFace一个开关类型的方块可以固定的方块面.
    Shape铁轨方块可使用的不同形态.
