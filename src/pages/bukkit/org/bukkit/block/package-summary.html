Package org.bukkit.block

用于操作在世界内的三位像素(方块)，包括一些特定的状态.

See: Description

  • Interface Summary 
    Interface Description
    Banner代表旗帜.
    Barrel代表木桶.
    Beacon代表信标.
    Bed代表床.
    Beehive代表蜂巢.
    Bell代表钟.
    BlastFurnace代表高炉.
    Block代表一个方块.
    BlockState表示方块某一时刻的状态的快照, 不会随方块本身的变更自动更改.
    BrewingStand代表酿造台.
    BrushableBlockRepresents a captured state of suspicious sand or gravel.
    CalibratedSculkSensorRepresents a captured state of a calibrated sculk sensor
    Campfire代表营火.
    Chest代表箱子.
    ChiseledBookshelfRepresents a captured state of a chiseled bookshelf.
    CommandBlock代表命令方块.
    Comparator代表关闭或打开状态的红石比较器.
    Conduit代表潮涌核心.
    Container代表容器方块.
    CreatureSpawner代表刷怪笼.
    DaylightDetector代表(反向)阳光探测器.
    DecoratedPotRepresents a captured state of a decorated pot.
    Dispenser代表发射器.
    Dropper代表投掷器.
    EnchantingTable代表附魔台.
    EnderChest代表末影箱.
    EndGateway代表末地折跃门.
    EntityBlockStorage代表可存储实体的方块 (译注:或者称为实体容器方块).
    Furnace代表熔炉.
    HangingSignRepresents a captured state of a hanging sign.
    Hopper代表漏斗.

    SculkShriekerRepresents a captured state of a sculk shrieker.

    Sign代表告示牌或者墙上告示牌.
    Skull代表头颅.
    Smoker代表烟熏炉.
    Structure代表结构方块, 其可从文件中加载方块或保存方块到文件.
    SuspiciousSandRepresents a captured state of suspicious sand.
    TileState代表持有指定位置的 tile entity 的方块状态.
