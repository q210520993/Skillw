      • setConsumeItem

         void setConsumeItem(boolean consumeItem)

        Set whether or not the consumable item should be consumed in this event. If set to false, it is recommended that a call to updateInventory is made as the client may disagree with the server's decision to not consume a consumable item.

        This value is ignored for entities where items are not required (skeletons, pillagers, etc.) or with crossbows (as no item is being consumed).

        Parameters:
        consumeItem - whether or not to consume the item
