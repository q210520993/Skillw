Class EntityPotionEffectEvent

  • All Implemented Interfaces:
    org.bukkit.event.Cancellable 
    
public class EntityPotionEffectEvent
extends EntityEvent implements Cancellable

    当实体上的药水效果发生改变时触发本事件.

    如果此事件被取消, 将不会对实体做出改变.
