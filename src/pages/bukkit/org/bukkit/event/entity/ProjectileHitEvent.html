      • setCancelled

         void setCancelled(boolean cancel)

        Whether to cancel the action that occurs when the projectile hits. In the case of an entity, it will not collide (unless it's a firework, then use FireworkExplodeEvent). In the case of a block, some blocks (eg target block, bell) will not perform the action associated. This does NOT prevent block collisions, and explosions will still occur unless their respective events are cancelled.

        Parameters:
        cancel - true if you wish to cancel this event
