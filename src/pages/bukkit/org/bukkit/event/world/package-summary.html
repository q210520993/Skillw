Package org.bukkit.event.world

与世界状态和变化有关的事件.

See: Description

  • Class Summary 
    Class Description
    AsyncStructureSpawnEventCalled when a Structure is naturally generated in the world.
    ChunkEvent代表与区块有关系的事件.
    ChunkLoadEvent当一个区块被加载时调用.
    ChunkPopulateEvent当一个新生成的区块填充完毕时调用.
    ChunkUnloadEvent当一个区块被卸载时调用
    EntitiesLoadEvent当实体将被加载时调用.
    EntitiesUnloadEvent当实体被卸载时调用.
    GenericGameEvent代表 Mojang 通用游戏事件.
    LootGenerateEvent当生成给予InventoryHolder的战利品时调用.
    PortalCreateEvent当传送门被创建时调用.
    SpawnChangeEvent一个在世界的出生点被改变时调用的事件.包含这个世界之前的出生点.
    StructureGrowEvent当一个有机结构试图生长的时调用此事件(树苗 -> 树, 蘑菇 -> 巨型蘑菇, 自然生长的或使用骨粉生长的).
    TimeSkipEvent当世界内的世界发生快进时调用.
    WorldEvent代表世界范围内的事件.
    WorldInitEvent当一个世界被初始化时调用.
    WorldLoadEvent当一个世界被加载时调用.
    WorldSaveEvent当一个世界被保存时调用.
    WorldUnloadEvent当一个世界被卸载时调用.
