Class GameEvent

  • All Implemented Interfaces:
    org.bukkit.Keyed 
    
public final class GameEvent
 implements Keyed

    代表 Mojang 通用游戏事件.

    译注:这些事件非 Bukkit 框架内提供的事件, 而是 Mojang 在原游戏引入的事件, 目前已知原版游戏用于Sculk_Sensor中.

    不过 Bukkit 框架提供了一个新的事件: GenericGameEvent, 使得监听 Mojang 游戏事件成为可能.

    原文:Represents a generic Mojang game event.
